使用root 用户用 vi 或 vim 编辑/etc/sudoers文件，找到 "root ALL=(ALL) ALL"一栏，在下一行填写：

```
user ALL=(root) NOPASSWORD:yum *
```

然后按：wq!完成编辑。注意这里要用到 ! 强制保存，如果不用的话会提示这是一个readonly文件，不支持修改。
